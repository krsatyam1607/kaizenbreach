## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18+), or  
- Bun (recommended)

---

### Installation

```bash
git clone https://github.com/krsatyam1607/kaizenbreach.git
cd kaizenbreach
````

Install dependencies:

```bash
npm install
# or
bun install
```

Run the dev server:

```bash
npm run dev
# or
bun run dev
```

Open in browser:

```
http://localhost:8080
```
